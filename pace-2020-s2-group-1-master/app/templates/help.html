General Instructions and Link to User Guide

Initial Setup

When the project is initially opened, there will be no database with which to run queries on.

To rectify this, click on the Upload tab to submit a .csv file with which to populate the database

Running Queries

Once the system has a database, navigate to the Search tab and enter your queries in the provided bar

To get a better idea of what queries you can run, consider the Data Breakdown section of the Search page or click on the Current Data tab
