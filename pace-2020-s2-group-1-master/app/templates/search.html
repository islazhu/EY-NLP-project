Guide to Data Types

Below is an example of the most common data types in SQL. For a more detailed guide on data types click here
  • VARCHAR or TEXT: This column stores text as a value. Example ("John", "Red", "2m41fs6")
  • BIGINT: This column stores a number value less than 19 digits in length. Example (500,000,000,000)
  • INT or INTEGER: This column stores a number value less than 10 digits in length. Example (100)
  • FLOAT: This column stores a number with a decimal point up to 16 digits in length as value. Example (3.14159265359, 1.0, 1.1)
  • REAL: This column stores a number with a decimal point up to 6 digits in length as value. Example (3.14159265359, 1.0, 1.1)
  • BOOLEAN: This column stores either a 0 or 1 value to represent true or false. Example (0,1)
  • DATE: This column stores day, month and year values. Example (05-12-2019)
  • TIME: This column stores hour minute and second values. Example (2:20:42)
